Reporte interactivo de rendimiento y conversión (Google Analytics & Looker Studio)

Vincular Reporte de Looker Studio

Aún no se ha configurado el enlace de visualización. Puedes integrar de manera segura cualquier reporte de Looker Studio conectado a tu propiedad de analítica para visualizarlo aquí directamente.

Pasos para Obtener tu Enlace de Inserción:

  1. Ingresa a Google Looker Studio y abre tu reporte de rendimiento de Google Analytics 4.
  2. En la esquina superior derecha, haz clic en el botón Compartir (junto al menú de Editar).
  3. Selecciona la opción Insertar informe (Embed report).
  4. Asegúrate de marcar la casilla de Habilitar inserción y selecciona la pestaña Insertar URL.
  5. Copia el enlace URL que se te proporciona (debe comenzar con https://lookerstudio.google.com/embed/...).
  6. Presiona el botón de Configurar Reporte arriba en esta página y pega el enlace.

Configurar Enlace de Reporte

El enlace debe comenzar con https://lookerstudio.google.com/embed/... Si pegas el código HTML iframe completo (ej: <iframe src="..."></iframe>), el sistema extraerá automáticamente el enlace src por ti.
